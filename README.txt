To build on Linux, run "build" file.
To install on Linux (which just copies the output binary) to a directoy, run "install" file.
To uninstall on Linux, just delete the binary in whatever directory you installed it in. The "uninstall" file will just tell you this.

To build on Windows, open the project folder in a Visual Studio developer console instance and run build_on_windows.bat.
To install on Windows, just copy nc.exe somewhere that's in your path. You're on your own for this one.
To uninstall on Windows, just delete nc.exe from where ever you ended up putting it.
